# PLED: A Comprehensive Pipeline for Electric-Field-Guided Transcranial Magnetic Stimulation Targeting and Optimization

## 1. 安装

- `conda create -n monai_efield python=3.8.20`
- `conda activate monai_efield`
- `pip install numpy==1.21.4`
- `pip install torch==1.13.0+cu117 torchvision==0.14.0+cu117 torchaudio==0.13.0 --extra-index-url https://download.pytorch.org/whl/cu117`
- `pip install monai==1.3.0`
- `pip install -r requirements.txt`

**Note**
This repository is actively being updated. Essential code for E-field estimation based on deep learning is provided; supplementary tools and documentation will be released progressively. Check back soon or watch this repository for updates.
